Gênero Textual


Os gêneros textuais são as diferentes fortmas de textos usadas para transmitir as mensagem para seus receptores.
Os gêneros textuais são classificados conforme a sua função comunativa, ou seja, sua estrutura e linguarem se diferem de
acordo com o tipo de texto que está sendo produzido.

tipos de textos:

Existem cinco tipos de textos:

  1. Narrativo;
  2. Descritivo;
  3. Expositivo;
  4. Argumentativo;
  5. Injustivo

Diferença entre tipo é genero
