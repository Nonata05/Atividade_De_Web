Bula de rémedio


Para que serve a bula do medicamento? A bula é um documento legal sanitário
que serve para obter informações e orientações sobre medicamentos necessárias
para o uso seguro e tratamento eficaz.

Na bula contem os seguintest topicos:

  1. Para que esse médicamento é indicado?
  2. Como esse medicamento funciona?
  3. Quando não devo usar este medicamento?
  4. O que devo saber antes de usar este medicamento?
  5. Onde, como e por quanto tempo posso guardar este medicamento?
  6. Como devo usar esse medicamento?
  7. O que devo fazer se eu esquecer de usar esse medicamento?
  8. Quais os males que esse medicamento pode me causar?
  9. O que fazer se alguem usar uma quantidade maior que a indicada desse medicamento?
